# SAST Analysis
## Используемый инструмент

Для статического анализа исходного кода был выбран инструмент **Semgrep**.

Semgrep — инструмент статического анализа исходного кода (SAST), позволяющий находить потенциальные уязвимости, ошибки безопасности и небезопасные конструкции в коде.


## Процесс выполнения проверки

Анализ выполнялся на исходном коде проекта DefectDojo.

Установка инструмента:

```bash
pipx install semgrep
```

Запуск анализа:

```semgrep scan \
  --config p/ci \
  .
```

## Интеграция с DefectDojo

Результаты анализа были сохранены в формате JSON и импортированы в DefectDojo.
В DefectDojo был создан Test типа Semgrep Scan, после чего результаты проверки стали доступны в системе управления уязвимостями.
