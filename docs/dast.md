# DAST Analysis
## Используемый инструмент

Для динамического анализа веб-приложения был выбран инструмент **OWASP ZAP**.

OWASP ZAP — инструмент для поиска уязвимостей в работающих веб-приложениях.


## Процесс выполнения проверки

Анализ выполнялся на запущенном веб-приложении DefectDojo.
Для проверки использовался режим baseline scan.


## Интеграция с DefectDojo

В DefectDojo был создан Test типа ZAP Scan, после чего результаты динамического анализа стали доступны для просмотра и анализа.
OWASP ZAP успешно использован для выполнения динамического анализа веб-приложения DefectDojo.
Результаты проверки были переданы в систему управления уязвимостями DefectDojo.
